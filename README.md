## 🌐 About

**CYBERLUX** is a multi-token staking platform on Monad Mainnet. Stake your favorite tokens, earn APR rewards, swap assets, and view your NFT cards — all in one cyberpunk-themed DeFi station.

- ⛓ Network: **Monad Mainnet** (Chain ID: 143)
- 🌐 App: **[cyberlux-station.pages.dev](https://cyberlux-station.pages.dev)**
- 🔧 Stack: React + TypeScript + Thirdweb + Cloudflare Pages + Workers + D1 + KV

## 🕹 Features

- **Multi Token Staking** — stake any supported token and earn APR rewards
- **Oracle Prices** — live Chainlink price feeds for BTC, ETH, MON, USDC
- **Buy with Card** — purchase tokens directly via fiat
- **Swap** — swap tokens within the platform
- **NFT Card Viewer** — view your NFT cards by Token ID
- **Ecosystem Panel** — explore the Monad ecosystem
